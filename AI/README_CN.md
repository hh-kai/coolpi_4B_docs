# AI

## 简介

RK3588-s中含有三个NPU核心，处理性能高达6T。
NPU的三个核心支持三核心合作，双核合作和独立工作三种应用场景。
能够有效的应对多任务多场景并行的应用场景。同时，它还支持主流的深度学习框架：TensorFlow、Tflite、Pytorch、Onnx等。
在本章节中你将学会如何使用RK3588的NPU快速部署AI程序。
